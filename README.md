# 📝 Task Manager (Simple CRUD App)

A basic Task Manager web application built using **HTML, CSS, and JavaScript**.
This project allows users to **Add, Edit, and Delete tasks** easily.

---

## 🚀 Features

* ➕ Add new task with description
* ✏️ Edit existing task
* ❌ Delete task
* 📋 Display all tasks dynamically

---

## 📂 Project Structure

```
Task-Manager/
│
├── index.html     # Main HTML file
├── style.css      # Styling file
├── script.js      # JavaScript logic
└── README.md      # Project documentation
```

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript 

---

## ⚙️ How It Works

1. User enters:

   * Task Name
   * Description

2. Clicks **Add Button**

3. Task is stored in an array and displayed in a list

4. User can:

   * Edit task ✏️
   * Delete task ❌
